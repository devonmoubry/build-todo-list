# Build a ToDo List

- Node.js
- Express.js

### Build Steps:

- `mkdir <project-name>`
- `cd <project-name>`
- `npm init`
- `gitignore node`
- `npm install`
- `npm install express --save`
- `npm install mustache mustache-express --save`
- `touch README.md`
- `touch app.js`
- `mkdir views`
- `cd views`
- `touch todo.mustache`
